Managing Watches

Adding a watch:

To add a watch, select the Add tab in the Watches window. This tab contains a form with all the details you need to specify for a watch, laid out in the form of sentences.
If you want to watch a specific memory address, first select byte or word from the combo box provided. Next specify the address of the start of that memory location (must be a valid hexadecimal word). Then click Add.
If you want to a watch a variables that your code defines. Select that variable from the combo box provided and click Add.

After you add the desired watch, you are automatically taken back to the Current watches tab. You can see that the new watch has been added to the list of watches, and is currently selected.
 

Removing watches:

To remove one or more watches, select the Current watches tab in the watches window. Select one or more watches from the list (you can multi-select by holding down Shift or Ctrl on WIN NT platforms). The selected watches are highlighted in blue. Once you're satisfied with you selection, click Remove and they're gone.
 

Note:

You can only add/remove watches when your program is idle (i.e. not running).
