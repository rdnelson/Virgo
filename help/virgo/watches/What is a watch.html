What is a watch?

A watch is probe that enables you to monitor changes in a specific location in memory.

For example, let's say your program places important information at memory address 223AH, and let's say that you need to know what that memory cell contains at all times during execution. Instead of enabling the memory window and watching the changes to that specific cell, you can tell the debugger that this memory cell is critical and therefore needs a watch to be placed on it. This watch will then appear in a separate window, telling you the value of that memory cell at all times.
