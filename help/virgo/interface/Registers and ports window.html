The registers and ports window

This window is where you can look at what happens inside the P86 processor. As you can see above, you have three separate views:

Registers:

Here you can watch the value inside all P86 registers at any instant of time. Every register has its value as the label of a separate button. Click on a button to change the value of the associated regsiter. The new value has to be a valid hexadecimal word.
More about the P86 registers.


Ports:

Here you can watch the values of important status/data ports. You can't change the values of ports (except using the OUT instruction).
More about the P86 ports.
Flags:
Every flag is represented by a checkbox. If the checkbox is checked, the flag is set (equal to one). Otherwise the flag is not set (equal to zero). To change the value of a flag, check/uncheck the associated checkbox.
More about the P86 flags.
