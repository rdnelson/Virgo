The memory window

The memory window enables you to view the contents of all memory cells at once. It is a very helpful tool. However, it devours your system resources so that performance will degrade considerably if this window is open. That's why this window is omitted by default. If you believe in the power of the machine you're running Virgo on, you can always enable the memory window:

  • In the main window, select the windows menu.
  • check the checkbox that reads "memory window enabled"
Once the memory window is enabled, you will notice a difference in Virgo's performance. To disable this window, simply close it.

Every memory address consists of four hex digits. The most significant three are listed as row entries in the first column titled "Address". The least significant hex digit is used as a column title. For example, to read the value of the memory cell 003A, look in the row that starts with 003, under the column titled A. (See the figure above)

The memory window also enables you to modify the contents of any memory cell. Double click on the desired cell and a dialog will appear asking you to enter the new value. The new value must be a valid hexadecimal byte or else it will be ignored.
