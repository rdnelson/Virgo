The watches window

This window helps you view and manage your watches. I assume you know what a watch is. The watches window consists of two tabs:

Current watches tab:

This tab shows you the current value of every watch that you have set. This information is updated constantly so that the values are correct at any instant of time. Byte watches have 8-bit long values, while word watches have 16-bit long values.

The Remove button will remove all the watches that are selected. The currently selected watches are highlighted in blue. You can multi-select any desired combination of watches to remove them.

Add tab:


This tab allows you to add any kind of watch.

Fill out the first sentence in order to add an address watch. The memory address you specify must be a valid hexadecimal word. Press Add when you're done.

Fill out the second sentence in order to watch a certain variable. The combo box provided contains a list of all the variables that you declared in your code. Choose the desired variable name and press Add.

When you add a watch, you are automatically taken to the Current watches tab, and the newly added watch is automatically highlighted.
