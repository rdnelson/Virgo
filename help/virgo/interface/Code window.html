The code window

This window shows you details about the code that your OBJ file contains. Every row corresponds to one assembly instruction in your source code (which you originally wrote in you ASM file).

You can select an instruction by clicking in the appropriate row. The selected instruction is outlined in blue.

One of the instructions is higlighted in red. This tells you which instruction is the current instruction. The current instruction is the instruction that IP currently points to. You can't specify which instruction is the current one, unless you manually set the value of IP in the registers and ports window.

Every row contains four entries:

Breakpoint indicator:

This checkbox indicates whether this instruction is a breakpoint or not. Double click it to set/reset a breakpoint. You can also set breakpoints in the breakpoints window.
Start address:
This field contains the memory address where the first byte of the instruction is stored.
Label:
This field contains the label that you gave for the instruction (if any). If your source code gives multiple labels for the same instruction, only one of them is displayed.
Source code:
This field contains the assembly code associated with the instruction, along with any comments that appeared in you ASM file.
