Loading your OBJ file

To load an OBJ file into Virgo, select the Open item from the File menu. A dialog appears enabling you to browse and select the desired OBJ file. You can also type the name and path of the file you wish to open in the File name text field. Once you have selected a file, click Open and that file will be loaded into Virgo.

Loading a file resets the state of the machine (memory, processor, screen etc.), loads the byte code and variables into their specific location in memory, and sets IP to point to the start of the code.

Note that Virgo can open one OBJ file at a time, therefore loading a file automatically closes the previously opened file.
