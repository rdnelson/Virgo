What is Virgo?

No, it's not one more way to torture engineering students! 

Virgo is simple debugger for assembly language programs. It is designed to help you understand how a computer system works, and how the assembly code that you write affects the state of a computer system.

As such, Virgo has windows that show you details about the current state of a simple machine (called P86), and other windows that allow you to load up OBJ files and control their execution.

Using Virgo:

Virgo is meant to be user friendly (or so I hope). Here is a list of tasks that a Virgo user might want to do:
 
 
Loading your OBJ file
Debugging your program
Modifying the state of the computer system
Managing watches
Managing breakpoints

Assembling:

Unfortunately, the current version of Virgo does NOT assemble your ASM files for you. You need an assembler for this task.
